# gamezone_orders_analysis_project
Data analytics project exploring 22K Game-Zone orders (2019–2022) with Excel cleaning and interactive Power BI dashboards. Key sales trends, product performance, and actionable business insights.

## Overview  
Analyzed 22,000+ GameZone e-commerce orders (2019–2022) for product, revenue, and channel trends. Built interactive Power BI dashboards using cleaned Excel data with cross-page navigation.

## Dataset  
Order-level data: purchase date, product, price, region, and marketing channel.

## Tools Used  
- **Excel:** Data cleaning  
- **Power BI:** Dashboard design & navigation

## Steps  
1. Data cleaning and standardization (Excel)  
2. Exploratory analysis  
3. Power BI dashboard creation  
4. KPI cards and navigation setup

## Dashboard  
Interactive visualizations for product performance, revenue breakdown, region trends, and drill-through analysis.

## Results  
Identified top products, seasonal peaks, and data-driven business recommendations.

## How to Run  
1. Download repo & dataset  
2. Open the Power BI file  
3. Use slicers/navigation to explore insights
